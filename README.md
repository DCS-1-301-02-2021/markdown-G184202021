# 情報工学科の紹介
#### コンピュータの基礎から、高度なプログラミングまで

*情報化社会をさらに進化させる、IT業界の新たな**リーダー**を育成*
*実社会に役立つ総合的能力を修得します。*
### 年間の流れ
1. 大学の学びや生活に慣れましょう
2. コースを選択し、より深く学んでいきます
3. 専門科目が増えていきます。また後期には卒業後が見え始めます
4. 研究室に配属して卒業研究に取り組みます
